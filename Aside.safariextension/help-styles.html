Styles

Each Cloudmarks style is a CSS stylesheet with rules that control the appearance of the Cloudmarks popup and bookmark list. If you know CSS—and even if you don't—feel free to change whatever you like. I recommend editing a duplicate of a default style: that way, if you mess it up, you won't have to click Restore Defaults and lose all changes to all styles (including losing any new styles).

If you don't know CSS but would like to learn, a good starting point is the W3C's basic CSS Tutorial.

Please note, styles have no effect when Cloudmarks is using a popover. They only apply when using the iframed popup.
