Setting the hotkey

To set the hotkey that invokes the Cloudmarks popup, click inside the input box and press the desired key combination. The hotkey is saved immediately and can be used right away.

The hotkey can be any character key, with or without modifiers such as Command, Option, and Shift. By "character key" we mean any key that produces a visible character if you press it while typing text. (This excludes special keys such as PageDown, Home, NumLock, the arrow keys, etc.)

If you enter a key combination involving the Shift and/or Option keys, what appears in the input box is the character that would appear if you typed that key combination normally. So, for example, if you press Option-Y, you will see the character ¥ in the input box, not the words "Option-Y"; but your hotkey will in fact be Option-Y.

Note: If you enter a hotkey that invokes a built-in Safari command, such as Command-P, Cloudmarks will save the hotkey, but it won't prevent Safari from executing the built-in command.

Settings for opening bookmarks

Open bookmark in new tab
If this option is selected, the highlighted bookmark will open in a new tab instead of the current tab when you press Enter.

Note: You can temporarily reverse this setting by holding down the Command key when pressing Enter.

Open new tab in background
If this option is selected, when opening a bookmark in a new tab, the new tab will be opened in the background. Otherwise, the new tab will become active upon opening.

Note: You can temporarily reverse this setting by holding down the Shift key when pressing Enter.

Position new tab next to current tab
If this option is selected, when opening a bookmark in a new tab, the new tab will be placed next to the current tab. Otherwise, it will be placed at the end.

Use new window instead of new tab
If this option is selected, when opening a bookmark in a new tab, the bookmark will open in a new window instead.

Note: If this option is selected, the previous two options have no effect.
